# PlantUML Gallery

제가 보려고 만든, PlantUML Gallery 입니다.

PlantUML 다이어그램이 잘 동작하는지도 테스트하고 있습니다.

잘 나오네요.

@startuml
actor user
component world
user -> world: "Hello, world!"
@enduml

위 코드를 중간에 삽입하면 됩니다.
